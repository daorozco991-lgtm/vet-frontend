# Huellitas Frontend

Aplicación web frontend para la gestión clínica veterinaria, construida con React, TypeScript y Vite.

## Nota

Este proyecto debe conectarse al backend [**Vet Backend
**](https://github.com/daorozco991-lgtm/vet-backend.git)
Asegúrese de tener el backend ejecutándose antes de iniciar la aplicación.

## Características

- Autenticación mediante JWT.
- Gestión de dueños.
- Gestión de mascotas.
- Gestión de citas veterinarias.
- Búsqueda rápida de registros.
- Diseño responsive.
- Validaciones de formularios.

## Requisitos

- Node.js 18+
- npm o yarn

Siga las instrucciones a continuación para ejecutar el proyecto localmente.

## Instalación

1. Clone o descargue este repositorio.

```bash
git clone https://github.com/daorozco991-lgtm/vet-frontend.git
```

2. Instale las dependencias:

```bash
npm install
```

## Ejecutar el proyecto en desarrollo

Desde la raíz del proyecto ejecute:

```bash
npm run dev
```

La aplicación estará disponible en `http://localhost:5173`

## Iniciar sesión

haga click en registrarse y cree un usario, luego inicie sesión para acceder a la aplicación.
